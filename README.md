# 🚀 HELPER VPN

**HELPER VPN** is a powerful **Ubuntu 24.04 LTS** auto installer that deploys a complete VPN server with SSH, WebSocket, Xray, Stunnel, Dropbear, BadVPN and many advanced management features in just a few minutes.

---

## ✨ Features

- ✅ Ubuntu 24.04 LTS Support
- ✅ OpenSSH
- ✅ SSH WebSocket (TLS & Non-TLS)
- ✅ Dropbear (Ports 109 & 143)
- ✅ Stunnel5 (Ports 447 & 777)
- ✅ Xray VMess WebSocket
- ✅ Xray VLESS WebSocket
- ✅ Xray Trojan WebSocket
- ✅ BadVPN UDPGW (7100 / 7200 / 7300)
- ✅ Automatic Let's Encrypt SSL
- ✅ BBR TCP Congestion Control
- ✅ Fail2Ban Protection
- ✅ Auto Expire Accounts
- ✅ Auto Reboot
- ✅ Multi Login Killer
- ✅ DNS Changer
- ✅ Backup & Restore
- ✅ Bandwidth Monitor
- ✅ RAM Monitor
- ✅ Full VPS Management Menu
- ✅ Easy SSH & Xray Account Management

---

# 📦 Supported Operating System

- Ubuntu 24.04 LTS (Recommended)

# ⚡ Installation

Run the command below as **root** on a fresh Ubuntu 24.04 VPS.

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/helperxvpn/HELPERVPN/main/HELPER_VPN.sh)
```
